# Шаблоны документов EduCase

На основе реальных ведомственных документов. Каждый шаблон = поля + ожидаемые значения для
проверки. Документ выбирается из списка с обманками, затем заполняются поля.

## Документы (основа)
- **Приложение 1 — ДМ4 / Внеочередное донесение** с критериями вспышки.
- **Приложение 3 — Ежемесячный отчёт.**
- **Приложение 22 — оценка СЭО**: четыре уровня + числовые пороги.
- **Приложение 23 — формулы расчёта заболеваемости.**
- **Акт расследования вспышки** — 11+ полей.
- Прочее по этапам: форма 23, акт ГСЭН, донесение командира (с обманками), план и приказ (с обманками).

## Механика проверки
- Выбор правильного документа из списка (есть документы-обманки).
- Заполнение полей → сверка с ожидаемыми значениями (точное/числовое сравнение по правилам поля).
